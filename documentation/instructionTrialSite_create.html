Create a new trial site

A user with the right "Administrator" is able to create a new trial site. Therefore, it is necessary to select the menu item "Trial site" -> "Add". Afterwards it is possible to enter the trial site information in two steps.



In the first step general information (e.g. name, street, ...) are requested. With click at the tab "Password" or at "Next" the last step of the trial site creation process is show.



With the chosen password it is for new user possible to use the self registration process. To finish the creation process it is necessary to click the button "save".
